# auth-admin

React + TypeScript 기반 SPA 프로젝트입니다.

## 시작하기

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

## 기술 스택

- React 18
- TypeScript 5
- Vite
- Zustand
- AG-Grid
- Recharts
